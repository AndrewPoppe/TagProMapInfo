Make Your Own

Make your own map metrics here. Type a name in the "Name" input box and then a formula in the "Formula" box.
You can reference other metrics by using a $ followed by the ID of the metric you want to use.
So, if I wanted to make a metric that adds all goal tiles together and divides by area, I would type:

($31 + $32) / $3

Once you've made your own, you can reference those too. Just make sure not to delete any formulae that are referenced by other formulae or there will be errors.
